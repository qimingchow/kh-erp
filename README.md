# 坤禾半导体 ERP 原型

这是一个前端静态原型，用于演示坤禾半导体的来料录入、库存管理、出库记录、生产计划、机台看板和财务记录流程。

## 本地预览

```bash
python3 -m http.server 4173
```

打开：

```text
http://127.0.0.1:4173/index.html
```

不要直接用 `file://` 打开，ES Module 在文件路径下可能无法稳定加载。

## 测试账号

- 管理员：`admin / admin123`
- 录单人员：`clerk / clerk123`

## 当前权限

- 管理员：全部功能权限，可删除来料单。
- 录单人员：可查看、新增、编辑来料单，不可删除。
